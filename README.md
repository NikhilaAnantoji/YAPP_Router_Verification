YAPP Router Verification (UVM)

This project contains a complete UVM-based verification environment for the YAPP Router.
It includes agents, sequence libraries, monitors, scoreboard, reference models, and testbench setup across multiple lab modules.

Features

UVM environment for YAPP Router

TX/RX agents with sequencer, driver, monitor

Packet-level verification using yapp_packet

Functional coverage and assertions

Multiple test scenarios

Modular testbench structure for all lab exercises
